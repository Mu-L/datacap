---
title: 分页实体
---

本文档主要是用于描述分页配置，所有涉及到分页数据的 API 都是基于该实体返回。

## Body

=== "示例"

    ```json
    {
        page: 1,
        size: 10
    }
    ```

=== "参数"

    |参数|类型|描述|默认值|
    |---|---|---|---|
    |`page`|数值|当前页码|1|
    |`size`|数值|每页条数|10|

## Response

=== "示例"

    ```json
    {
      "page": 1,
      "size": 12,
      "total": 3,
      "totalPage": 1,
      "content": []
    }
    ```

=== "参数"

    |参数|类型|描述|
    |---|---|---|
    |`page`|数值|当前页码|
    |`size`|数值|每页条数|
    |`total`|数值|总条数|
    |`totalPage`|数值|总页数|
    |`content`|数组|当前页数据|
