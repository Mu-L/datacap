- [Query] 修复并发访问导致无法记录查询历史记录的问题
- [Dashboard] 支持上传缩略图
- [Dashboard] 添加本地存储路径
- [User] 修复用户登录注册加载状态
- [User] 支持后台添加用户
- [User] 优化用户操作函数
